Setting Up SSH Keys for GitHub

GitHub allows you to securely authenticate with your account using SSH keys. Follow these steps to set up SSH keys:

  1. Generate an SSH key pair on your local machine:

    
# Replace your_email@example.com with your email
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

  2. Add your SSH key to the SSH agent:

    
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa # Replace id_rsa with your key file if it's different

  3. Copy your public key to your clipboard:

    
# On Windows, you can use 'clip' command. On Linux or macOS, use 'pbcopy'.
clip < ~/.ssh/id_rsa.pub

  4. Visit your GitHub account settings and add the copied public key to your SSH keys:

    
1. Log in to GitHub.
2. Click on your profile picture in the top right and select "Settings."
3. In the left sidebar, click "SSH and GPG keys."
4. Click "New SSH key" or "Add SSH key."
5. Paste your public key into the "Key" field.
6. Add a descriptive title (e.g., "My SSH Key") for your key.
7. Click "Add SSH key."

Your SSH key is now set up for use with GitHub, allowing you to securely authenticate and interact with your repositories.
